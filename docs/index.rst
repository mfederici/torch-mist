=================
Torch Mist Documentation
=================

Documentation for the ``torch_mist`` mutual information estimation package.
Code and additional information can be found in the `Github repository`_.

The documentation includes a `quickstart`_ guide, detailed `estimator tutorials`_ showcasing the estimators, and practical `use-case examples`_.

.. _Github repository: https://github.com/mfederici/torch-mist
